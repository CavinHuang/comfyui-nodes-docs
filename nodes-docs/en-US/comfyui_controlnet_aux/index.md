# comfyui_controlnet_aux
comfyui_controlnet_aux is a collection of plug-and-play node sets designed for use with ComfyUI, aimed at generating ControlNet hint images for various preprocessing tasks such as line extraction, depth estimation, and semantic segmentation. It integrates code from ControlNet's annotator folders and connects to the ðŸ¤— Hub for easy access to pre-trained models, supporting a wide range of preprocessors for creating hint images like stickman, canny edge, and more.
