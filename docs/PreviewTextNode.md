# Input types

    - Comfy dtype: STRING
    - Python dtype: str

# Output types
- string
    - 处理后的文本作为输出的一部分返回，封装了预览的文本。
    - Comfy dtype: STRING
    - Python dtype: Tuple[str]
- ui
    - 输出包括一个用户界面组件，用于显示输入文本的预览。
